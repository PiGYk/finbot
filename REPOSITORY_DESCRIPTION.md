# Repository Description

## Короткий опис для GitHub About

AI-powered Telegram bot for personal and family finance tracking with Firefly III, natural-language transaction parsing, receipt OCR/understanding, reminders, budgets, reports, and multi-profile support.

---

## Короткий опис українською

Telegram-бот для фінансового обліку з Firefly III та Claude: витрати, доходи, перекази, чеки, нагадування, бюджети, звіти, поради і підтримка кількох профілів.

---

## Розширений опис

Цей проєкт перетворює Telegram на простий інтерфейс для ведення фінансів. Користувач пише звичайною мовою, а бот розуміє намір, розбирає суму, категорію, рахунок і дату, після чого записує транзакцію у Firefly III.

Окрім звичайних текстових транзакцій, бот підтримує аналіз фото чеків через Claude, локальні правила категоризації, щоденні нагадування, бюджетні плани, базові звіти, фінансові поради та режим кількох профілів обліку.

Проєкт підходить для особистого використання, сімейного бюджету або невеликих окремих контурів обліку, де потрібен швидкий Telegram-first UX без важкого фронтенду.

---

## Основна цінність проєкту

- не треба відкривати Firefly III для кожної дрібної витрати
- можна вести облік прямо з Telegram
- є розбір тексту природною мовою
- є аналіз чеків з підтвердженням перед записом
- можна працювати з кількома профілями
- дані про транзакції зберігаються у Firefly III, а службовий стан локально

---

## Для кого цей проєкт

- для особистого фінансового обліку
- для сімейного бюджету
- для користувачів Firefly III, які хочуть швидкий мобільний інтерфейс через Telegram
- для внутрішніх pet-проєктів, де важлива швидкість інтеграції, а не академічна стерильність
